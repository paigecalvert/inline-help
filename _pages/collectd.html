--- id: collectd --- collectd is a free, open source system statistics collection daemon that monitors and stores data on the system on which it is running. Collectd is equipped with several plugins that allow you to monitor a wide range of metrics and perform a variety of tasks. By using collectd's Write HTTP plugin, you can begin monitoring your collectd data with Netuitive. For more information about collectd, see the following [documentation](https://collectd.org/).
